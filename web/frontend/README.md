# Frontend README

## Next.js Frontend for Silent Meeting Summarizer

### Setup

```bash
cd web/frontend
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:3000`

### Build

```bash
npm run build
npm start
```

### Testing

```bash
npm test
```
